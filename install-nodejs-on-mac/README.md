Install Node
----

```
brew install node
```

After node is installed, you should have `node` and `npm` installed in your machine.

Install Node Version Manager
----

Clone the NVM repository

```
git clone git://github.com/creationix/nvm.git ~/.nvm
```

Add to ~/.bash_profile

```
. ~/.nvm/nvm.sh
```
